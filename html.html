O que é HTML?

HTML significa HyperText Markup Language. É a linguagem padrão para criar páginas web.

Uso de Cabeçalhos

Os cabeçalhos são definidos com as tags <h1> a <h6>, onde <h1> é o cabeçalho mais importante e <h6> é o menos importante.

Parágrafos

Os parágrafos são definidos com a tag <p>.

Exemplo de Parágrafo

Este é um exemplo de parágrafo.

Listas Ordenadas

As listas ordenadas são definidas com a tag <ol> e os itens da lista com a tag <li>.

Exemplo de Lista Ordenada

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas

As listas não ordenadas são definidas com a tag <ul> e os itens da lista com a tag <li>.

Exemplo de Lista Não Ordenada

  • Item A
  • Item B
  • Item C

Tabelas

As tabelas são definidas com a tag <table>, as linhas com a tag <tr> e as células com as tags <th> (cabeçalho) ou <td> (dados).

Exemplo de Tabela

Nome Idade
Maria 30
João 25

Imagens

As imagens são definidas com a tag <img> e o caminho da imagem é especificado pelo atributo src.

Exemplo de Imagem

Links

Os links são definidos com a tag <a> e o destino do link é especificado pelo atributo href.

O que é CSS?

CSS significa Cascading Style Sheets. É uma linguagem usada para descrever a apresentação de um documento escrito em HTML ou XML.

Seletores Básicos

Um seletor básico seleciona elementos HTML com base em seu nome, classe ou ID.

        
        p {
            color: blue;
        }
        .classe-exemplo {
            font-size: 20px;
        }
        #id-exemplo {
            background-color: yellow;
        }

Propriedades e Valores

As propriedades são seguidas por dois pontos (:) e os valores por ponto e vírgula (;).

        
        body {
            background-color: #f0f0f0;
            margin: 0;
        }

Classes e IDs

Classes são definidas com um ponto (.) e IDs com uma hashtag (#).

Cascata e Especificidade

A cascata define como os estilos são aplicados, enquanto a especificidade define quais estilos têm precedência.
